# Experiment: quant-gen-0

## 市场
- BTC/USD (加密货币，7x24)
- 数据源: CoinGecko API (免费)

## 策略 (v2 — Ultra Scalper)
- **主策略**: EMA(2/5) + RSI(4) 超短线
  - 买入: EMA金叉 / RSI<25 / 看涨交叉
  - 卖出: EMA死叉 / RSI>75 / 看跌交叉
  - 平仓: RSI反向 / 信号反转 / 止损止盈
- **关键改进**: 从50%仓位降到20%，避免资金耗尽

## 回测结果 (7天小时线, 2026-02-20 ~ 2026-02-27)
| 策略 | 收益 | 交易数 | 胜率 |
|------|------|--------|------|
| **Ultra_2_5** | **+0.15%** | **44** | **56.8%** |
| Aggressive_3_7 | -0.87% | 35 | 54.3% |
| Scalper_3_8_wide | -0.94% | 33 | 45.5% |
| Scalper_3_8 | -1.36% | 28 | 50.0% |
| Fast_5_13 | -1.26% | 17 | 35.3% |

## 观察
- 超短EMA(2/5)在小时线上信号频繁，7天产生44笔交易
- 20%仓位是关键——50%仓位所有策略都归零
- 56.8%胜率 + 在下跌市微盈，策略有潜力
- 最大单笔亏损 -3.47%（止损触发），需要优化止损逻辑

## 模拟盘参数
- 本金: $10,000
- 单笔仓位: 20% capital
- 止损: -1.5%
- 止盈: +2.0%
- 检查频率: 每小时（hourly candles）
