# Experiment: quant-gen-0

## 市场
- BTC/USD (加密货币，7x24)
- 数据源: CoinGecko API (免费)

## 策略
- **主策略**: Combined EMA(9/21) + RSI(14) 过滤
  - 买入: EMA金叉 且 RSI < 50
  - 卖出: EMA死叉 或 RSI > 75
- **备选**: 纯EMA交叉（在趋势市表现更好）

## 回测结果 (90天, 2025-11-29 ~ 2026-02-27)
| 策略 | 收益 | 交易数 | 胜率 |
|------|------|--------|------|
| EMA 9/21 | -18.35% | 48 | 20.8% |
| EMA 12/26 | -20.62% | 36 | 19.4% |
| RSI 14 | -19.87% | 7 | 42.9% |
| Combined | -0.25% | 1 | 0.0% |

## 观察
- 过去90天BTC大幅下跌，纯趋势跟踪策略亏损严重
- Combined策略因RSI过滤减少了错误买入
- 需要加入做空能力或更好的止损

## 模拟盘参数
- 本金: $10,000
- 单笔仓位: 50% capital
- 止损: -3%
- 止盈: +5%
- 检查频率: 每小时
