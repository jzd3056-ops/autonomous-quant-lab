# Strategy: Combined EMA + RSI

## 逻辑
- **做多**: EMA(9) 上穿 EMA(21) 且 RSI(14) < 50
- **做空**: EMA(9) 下穿 EMA(21) 且 RSI(14) > 45
- **平多**: EMA死叉 或 RSI > 75 或 止损-3% 或 止盈+5%
- **平空**: EMA金叉 或 RSI < 22 或 止损-3% 或 止盈+5%
- **超买卖出**: RSI > 78 → 开空
- **超卖买入**: RSI < 22 → 开多

## 仓位管理
- 单笔: 50% capital
- 止损: -3%
- 止盈: +5%
- 最大持仓: 1个

## 回测表现 (90天下跌市)
- 收益: -0.25%
- 交易数: 1
- 评价: 保守但安全，在下跌市中几乎保本
