# 决策框架

## 交易循环
```
研究市场 → 构建假设 → 编写策略 → 回测 → 评估 → 执行/放弃 → 监控 → 复盘
```

## 决策记录格式
```markdown
### 交易决策 #N
- **时间**: 
- **标的**: 
- **方向**: 多/空
- **入场理由**: 
- **止损**: 
- **止盈**: 
- **仓位**: 占本金 %
- **结果**: 盈/亏 (金额/百分比)
- **复盘**: 
```

## 原则
1. **先回测再实盘** — 不经回测的策略不执行
2. **严格止损** — 每笔交易必须设止损
3. **仓位管理** — 单笔不超过本金 20%
4. **记录一切** — 每笔交易都要记录和复盘
5. **数据驱动** — 不凭感觉，看回测数据
